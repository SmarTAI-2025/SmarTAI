## 3.

给定递推关系 $a_n = 3a_{n-1} + 2a_{n-2}$，其中 $a_0 = 1, a_1 = 2$

特征方程：$r^2 = 3r + 2$，即 $r^2 - 3r - 2 = 0$

使用二次公式：$r = \frac{3 \pm \sqrt{9 + 8}}{2} = \frac{3 \pm \sqrt{17}}{2}$

设 $r_1 = \frac{3 + \sqrt{17}}{2}, r_2 = \frac{3 - \sqrt{17}}{2}$

通解为 $a_n = c_1 r_1^n + c_2 r_2^n$

利用初始条件：
- $a_0 = c_1 + c_2 = 1$
- $a_1 = c_1 r_1 + c_2 r_2 = 2$

解此方程组可得 $c_1$ 和 $c_2$ 的值。

```python
def fibonacci_number(n):
    # 边界情况处理
    if n <= 1:
        return n
    
    # 初始化前两个斐波那契数
    first = 0
    second = 1
    
    # 使用动态规划迭代计算
    for i in range(2, n + 1):
        third = first + second
        first = second
        second = third
    
    return second
```

## 7.

证明：在任何无向图中，度数为奇数的顶点个数必为偶数。

根据图论中的握手定理，图中所有顶点的度数之和等于边数的两倍。

设无向图有 $n$ 条边，则所有顶点度数之和为 $2n$，显然是偶数。

将顶点分为两类：
1. 度数为偶数的顶点
2. 度数为奇数的顶点

偶数度数顶点的度数之和仍然是偶数。

由于所有顶点度数之和为偶数，奇数度数顶点的度数之和也必须是偶数。

如果奇数度数顶点有 $k$ 个，每个顶点的度数都是奇数，那么 $k$ 个奇数的和为偶数当且仅当 $k$ 为偶数。

因此，度数为奇数的顶点个数必为偶数。
