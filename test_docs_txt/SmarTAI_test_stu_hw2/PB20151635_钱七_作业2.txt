```python
def fibonacci(n):
    if n <= 1:
        return n
    
    # 动态规划数组方法
    dp = [0] * (n + 1)
    dp[0] = 0
    dp[1] = 1
    
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```

## 7.

设无向图中奇度数顶点有k个，它们的度数分别为 $d_1, d_2, ..., d_k$。

根据握手定理：$\sum_{i=1}^{n} d_i = 2|E|$（偶数）

因为偶度数顶点度数之和也是偶数，所以 $\sum_{i=1}^{k} d_i$ 必为偶数。

由于每个 $d_i$ 都是奇数，只有偶数个奇数相加才能得到偶数，因此k为偶数。
