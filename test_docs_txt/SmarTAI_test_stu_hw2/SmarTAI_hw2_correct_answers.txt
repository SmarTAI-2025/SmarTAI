特征方程为 $x^2 - 3x - 2 = 0$，解得 $x_1 = \frac{3+\sqrt{17}}{2}, x_2 = \frac{3-\sqrt{17}}{2}$

通解为 $a_n = C_1 \left(\frac{3+\sqrt{17}}{2}\right)^n + C_2 \left(\frac{3-\sqrt{17}}{2}\right)^n$

代入初始条件：
$a_0 = C_1 + C_2 = 1$
$a_1 = C_1 \frac{3+\sqrt{17}}{2} + C_2 \frac{3-\sqrt{17}}{2} = 2$

解得 $C_1 = \frac{1+\sqrt{17}}{2\sqrt{17}}, C_2 = \frac{\sqrt{17}-1}{2\sqrt{17}}$

因此 $a_n = \frac{1+\sqrt{17}}{2\sqrt{17}} \left(\frac{3+\sqrt{17}}{2}\right)^n + \frac{\sqrt{17}-1}{2\sqrt{17}} \left(\frac{3-\sqrt{17}}{2}\right)^n$

```python
def fibonacci(n):
    if n <= 1:
        return n
    
    # 创建数组存储已计算的值
    dp = [0] * (n + 1)
    dp[0] = 0
    dp[1] = 1
    
    # 动态规划填表
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```

## 7.

设无向图 $G=(V,E)$ 有 $n$ 个顶点，度数分别为 $d_1, d_2, \ldots, d_n$。

根据握手定理：$\sum_{i=1}^{n} d_i = 2|E|$

即所有顶点度数之和等于边数的两倍，显然是偶数。

将顶点按度数奇偶性分组：
- 度数为偶数的顶点：其度数之和仍为偶数
- 度数为奇数的顶点：其度数之和必须为偶数（这样才能使总和为偶数）

设度数为奇数的顶点有 $k$ 个，每个顶点的度数为 $d_{i_1}, d_{i_2}, \ldots, d_{i_k}$，均为奇数。

因为奇数个奇数之和为奇数，偶数个奇数之和为偶数，所以要使 $\sum_{j=1}^{k} d_{i_j}$ 为偶数，必须 $k$ 为偶数。

因此，度数为奇数的顶点个数必为偶数。
