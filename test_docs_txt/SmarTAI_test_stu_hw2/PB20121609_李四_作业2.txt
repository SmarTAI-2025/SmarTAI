## 3.

对于递推关系 $a_n = 3a_{n-1} + 2a_{n-2}$，特征方程为 $x^2 - 3x - 2 = 0$

解得 $x = \frac{3 \pm \sqrt{9 + 8}}{2} = \frac{3 \pm \sqrt{17}}{2}$

通解为 $a_n = c_1(\frac{3 + \sqrt{17}}{2})^n + c_2(\frac{3 - \sqrt{17}}{2})^n$

由初始条件：
- $a_0 = c_1 + c_2 = 1$
- $a_1 = c_1(\frac{3 + \sqrt{17}}{2}) + c_2(\frac{3 - \sqrt{17}}{2}) = 2$

解得 $c_1 = \frac{1 + \sqrt{17}}{2\sqrt{17}}, c_2 = \frac{\sqrt{17} - 1}{2\sqrt{17}}$

## 5.

```python
def fib(n):
    if n <= 1:
        return n
    
    # 动态规划方法计算斐波那契数列
    prev2 = 0  # F(0)
    prev1 = 1  # F(1)
    
    for i in range(2, n + 1):
        current = prev1 + prev2
        prev2 = prev1
        prev1 = current
    
    return prev1
```

## 7.

在无向图中，所有顶点度数之和等于边数的2倍，这是一个偶数。

设奇度数顶点集合为S，偶度数顶点集合为T。

则 $\sum_{v \in S} deg(v) + \sum_{v \in T} deg(v) = 2|E|$

因为 $\sum_{v \in T} deg(v)$ 是偶数（偶数个偶数相加），$2|E|$ 也是偶数，

所以 $\sum_{v \in S} deg(v)$ 必须是偶数。

但S中每个顶点的度数都是奇数，奇数个奇数相加为奇数，偶数个奇数相加为偶数。

因此S中顶点个数必须是偶数。
