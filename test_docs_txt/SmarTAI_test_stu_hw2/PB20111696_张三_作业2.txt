递推关系 $a_n = 3a_{n-1} + 2a_{n-2}$ 的特征方程为 $x^2 - 3x - 2 = 0$

使用求根公式：$x = \frac{3 \pm \sqrt{9+8}}{2} = \frac{3 \pm \sqrt{17}}{2}$

所以 $x_1 = \frac{3 + \sqrt{17}}{2}, x_2 = \frac{3 - \sqrt{17}}{2}$

通解为 $a_n = C_1 x_1^n + C_2 x_2^n$

代入初始条件：
- $a_0 = C_1 + C_2 = 1$
- $a_1 = C_1 x_1 + C_2 x_2 = 2$

解这个方程组得到 $C_1$ 和 $C_2$ 的值。

```python
def fibonacci_dp(n):
    if n <= 1:
        return n
    
    # 使用动态规划数组
    dp = [0] * (n + 1)
    dp[0] = 0
    dp[1] = 1
    
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```

## 7.

证明：在任何无向图中，度数为奇数的顶点个数必为偶数。

设图中所有顶点的度数分别为 $d_1, d_2, ..., d_n$。

根据握手定理，所有顶点度数之和等于边数的两倍：
$$\sum_{i=1}^{n} d_i = 2|E|$$

因为边数是整数，所以度数之和是偶数。

将顶点按度数奇偶性分为两组：
- 偶度数顶点：度数之和为偶数
- 奇度数顶点：度数之和也为偶数（因为总和为偶数）

设奇度数顶点有 $k$ 个，每个的度数为 $d_{i_1}, d_{i_2}, ..., d_{i_k}$。

因为每个 $d_{i_j}$ 都是奇数，而它们的和是偶数，所以 $k$ 必须是偶数。

因此，度数为奇数的顶点个数必为偶数。
